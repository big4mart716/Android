# Android课程设计报告模板

## 一、项目概述

### 1.1 项目名称
个人名片应用（PersonalCard）

### 1.2 开发环境
- 开发语言：Java
- 开发平台：Android
- 最低支持版本：Android 5.0 (API 21)
- 目标版本：Android 13 (API 33)
- 开发工具：Android Studio
- 构建工具：Gradle

### 1.3 项目功能
本应用实现了一个简单的个人信息管理系统，主要功能包括：
1. 欢迎界面展示
2. 个人信息输入
3. 信息验证
4. 名片展示
5. 页面导航

## 二、需求分析

### 2.1 功能需求
- **FR1**: 应用启动后显示欢迎界面
- **FR2**: 用户可以输入个人信息（姓名、学号、专业、电话、邮箱）
- **FR3**: 系统验证信息完整性
- **FR4**: 以名片形式展示用户信息
- **FR5**: 支持返回主界面重新输入

### 2.2 非功能需求
- **NFR1**: 界面友好，符合Material Design规范
- **NFR2**: 响应速度快，操作流畅
- **NFR3**: 适配不同屏幕尺寸
- **NFR4**: 代码结构清晰，易于维护

## 三、系统设计

### 3.1 系统架构
本应用采用经典的MVC架构模式：
- **Model**: 数据通过Intent在Activity之间传递
- **View**: XML布局文件定义界面
- **Controller**: Activity类处理用户交互和业务逻辑

### 3.2 功能模块设计

#### 模块1：欢迎模块（MainActivity）
- 显示应用标题和描述
- 提供"开始创建名片"按钮
- 跳转到信息输入界面

#### 模块2：信息输入模块（InputActivity）
- 提供5个输入框（姓名、学号、专业、电话、邮箱）
- 验证输入完整性
- 将数据传递到展示界面

#### 模块3：信息展示模块（DisplayActivity）
- 接收并显示用户输入的信息
- 以卡片形式美化展示
- 提供返回首页功能

### 3.3 数据流设计
```
MainActivity → InputActivity → DisplayActivity
    |              |                |
    v              v                v
启动应用 → 输入信息 → 展示名片 → 返回首页
```

##四、详细设计

### 4.1 Activity设计

#### 4.1.1 MainActivity
**功能**: 应用入口，欢迎界面
**关键代码**:
```java
btnStart.setOnClickListener(new View.OnClickListener() {
    @Override
    public void onClick(View v) {
        Intent intent = new Intent(MainActivity.this, InputActivity.class);
        startActivity(intent);
    }
});
```

#### 4.1.2 InputActivity
**功能**: 信息采集和验证
**关键技术**:
- EditText组件实现信息输入
- Toast提示用户填写完整信息
- Intent传递数据到下一个Activity

**验证逻辑**:
```java
if (name.isEmpty() || studentId.isEmpty() || major.isEmpty() ||
    phone.isEmpty() || email.isEmpty()) {
    Toast.makeText(this, "请填写所有信息", Toast.LENGTH_SHORT).show();
    return;
}
```

#### 4.1.3 DisplayActivity
**功能**: 信息展示
**关键技术**:
- Intent接收数据
- CardView实现卡片效果
- FLAG_ACTIVITY_CLEAR_TOP实现返回栈清理

### 4.2 界面设计

#### 4.2.1 布局方式
- **LinearLayout**: 用于垂直排列组件
- **ScrollView**: 防止小屏幕显示不全
- **CardView**: 美化信息展示

#### 4.2.2 颜色方案
- 主色调：蓝色(#2196F3)
- 背景色：淡蓝色(#E3F2FD)
- 强调色：紫色(#6200EE)

#### 4.2.3 字体大小
- 标题：24sp
- 副标题：22sp
- 正文：18sp
- 按钮：18sp

## 五、实现技术

### 5.1 使用的Android组件

| 组件 | 用途 | 使用位置 |
|------|------|----------|
| Activity | 界面管理 | 所有界面 |
| Intent | 页面跳转和数据传递 | Activity之间 |
| TextView | 文本显示 | 所有界面 |
| EditText | 文本输入 | 输入界面 |
| Button | 按钮 | 所有界面 |
| Toast | 消息提示 | 输入验证 |
| LinearLayout | 线性布局 | 所有界面 |
| ScrollView | 滚动视图 | 输入界面 |
| CardView | 卡片布局 | 展示界面 |

### 5.2 关键技术点

#### 5.2.1 Activity生命周期
- `onCreate()`: 初始化界面和组件
- `onStart()`: Activity可见时调用
- `onResume()`: Activity获得焦点时调用

#### 5.2.2 事件处理
使用匿名内部类实现点击事件监听：
```java
button.setOnClickListener(new View.OnClickListener() {
    @Override
    public void onClick(View v) {
        // 处理点击事件
    }
});
```

#### 5.2.3 Intent传递数据
```java
// 发送数据
intent.putExtra("key", value);

// 接收数据
String value = getIntent().getStringExtra("key");
```

### 8.3 遇到的问题及解决方案

**问题1**: 初次运行时图标显示为默认Android图标
- **原因**: 未添加自定义应用图标
- **解决**: 使用系统默认图标，不影响功能

**问题2**: 信息展示页面返回后数据仍然保留
- **原因**: Activity未被销毁
- **解决**: 使用FLAG_ACTIVITY_CLEAR_TOP标志清理返回栈
